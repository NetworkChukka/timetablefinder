# 🎓 RUSL FAS Exam Timetable Finder

[![Live Site](https://img.shields.io/badge/Live-Site-brightgreen?style=for-the-badge)](https://networkchukka.github.io/timetablefinder/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

A lightweight, high-performance web application designed for students of the **Faculty of Applied Sciences (FAS)** at **Rajarata University of Sri Lanka**. This tool streamlines the process of finding and organizing examination schedules.

---

## ✨ Key Features

* **⚡ Smart Filtering:** Search by course code or subject name in real-time.
* **🌓 Adaptive Theming:** Seamlessly switch between Light and Dark modes (preferences are saved automatically).
* **🔢 Intelligent Sorting:** Course codes are sorted numerically (e.g., Level 1000 appears before Level 4000).
* **🖨️ Print-Ready:** Generates a professional, clean table layout optimized for A4 printing or saving as PDF.
* **📱 Fully Responsive:** Works perfectly on mobile devices for quick checks on the go.
